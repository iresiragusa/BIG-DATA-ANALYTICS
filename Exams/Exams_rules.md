## Exams rule

As you know Big Data is an integrated course which is made up of two modules, Big Data Tools and Big Data Analytics held during the first semester and the second semester respectively.

Which type of exams you are exected to have in June? It depends:

1. **Full-time students @ LM and Data LM-82 R**

   BIG-DATA C.I. -> Written and oral exam with topics of both Big Data Tools and Big Data Analytics
   *Note* if you passed the mid-term exam (Big Data Tools) the exam will follow the same rules as for Erasmus Students.
  
2. **Erasmus Students**

   BIG-DATA ANALYTICS -> Written and oral exam with Big Data Analytics topics.
   Be aware that some topics are pre-requisites from Big Data Tools, check the CourseIntro slides.

3. **PhD students**
   Oral presentation on the usage of Big Data Analytics topics in their reaseach

The exam for (1) and (2) will be divided into two steps, described as follow:

1. **Practical written exam**  

You will be asked to solve some exercises (like the ones we explored during the practical lessons) based on the concepts regarding the theoretical part of the course, using the techniques we explored in our practical lessons.
You will have access to both the theoretical slides and the documentation of the libraries you will need to solve the proposed exercises.

2. **Oral exam**

It will be an oral discussion about the theoretical topics of the course.
